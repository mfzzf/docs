---
title: "Platform API"
description: "Public API contract for workspace product resources."
---

The Platform API is the documented external contract for the workspace surfaces
in the Astraflow web app. It covers agents, environments, sessions, files,
skills, MCP servers, memory stores, credentials, deployments, API keys, usage,
request logs, workspace settings, and account profile endpoints.

<Info>
  The workspace is inferred from the API key. Use `Authorization: Bearer
  sk-af-...` with every request.
</Info>

The generated endpoint pages are sourced from
`api-reference/platform-openapi.json`.

Astraflow is a managed-agent platform for building agents, attaching skills and
files, running sessions in sandboxed environments, and streaming execution
events back to the product UI.

This documentation is intentionally split into two layers:

- **Guides** explain how the platform pieces fit together.
- **API Reference** is generated from the runtime OpenAPI schema.

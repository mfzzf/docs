Astraflow is a managed-agent platform for creating versioned agents, attaching
skills, files, MCP servers, memory, and credentials, then running those agents
as observable sessions in managed environments.

The documentation is split into two layers:

- **Guides** explain the product model and end-to-end workflows.
- **API Reference** is generated from OpenAPI for exact request and response
  shapes.
